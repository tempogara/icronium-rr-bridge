Start Here

Guida rapida per chi riceve il file distribuito di ICRONIUM RR Bridge e deve avviarlo per la prima volta.

Contenuto del pacchetto

  • app.jar
  • start.sh e stop.sh
  • start.bat e stop.bat
  • config/application.yml
  • logs/
  • data/
  • docs/

Requisiti

  • Java 21
  • Browser moderno
  • Accesso alla rete necessaria per Race Result

Primo avvio

  1. Scompatta il file zip
  2. Apri la cartella estratta
  3. Verifica Java con java -version
  4. Avvia l'applicazione

macOS / Linux

chmod +x start.sh stop.sh
./start.sh

Windows

Doppio clic su start.bat

Accesso

Dopo l'avvio apri il browser su:

http://localhost:8089

La porta è definita nel file config/application.yml.

Manuali

Nel pacchetto trovi:

  • docs/start-here-distribuzione.html
  • docs/manuale-utente.html
  • docs/manuale-tecnico.html

Per iniziare usa prima start-here-distribuzione.html, poi manuale-utente.html.

Log e dati

  • Log runtime: logs/app.out
  • Dati applicativi: data/

Stop

macOS / Linux

./stop.sh

Windows

Arresta il processo Java dal Task Manager oppure con taskkill.

Problemi comuni

  • Se il browser non apre la pagina, controlla logs/app.out
  • Se Java non è trovato, installa Java 21
  • Se la porta è occupata, modifica config/application.yml
