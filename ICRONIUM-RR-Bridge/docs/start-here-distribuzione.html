Start Here

Guida rapida per avviare il pacchetto distribuito di ICRONIUM RR Bridge.

Contenuto

Nel pacchetto trovi app.jar, script start/stop, configurazione, log, dati e documentazione.

Requisiti

Serve Java 21.

Avvio

macOS/Linux:

chmod +x start.sh stop.sh
./start.sh

Windows: usa start.bat.

Accesso

Apri il browser su http://localhost:8089, salvo diversa porta in config/application.yml.

Login

La home supporta RR online, RR locale e TZero.
