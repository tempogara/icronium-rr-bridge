Accesso

Apri il browser su http://localhost:8087.

La home presenta tre modalità: Login a RR online, Login a RR locale, Login a TZero.

Login

  • RR online: usa credenziali RR remote e le ricorda nel browser
  • RR locale: entra senza credenziali RR
  • TZero: chiede la root folder al primo accesso e poi la riusa
Dopo il login arrivi nella Race List.

Toolbar comune

Nelle viste operative trovi una navbar coerente con:

  • Open RR
  • RR Live
  • TP View
  • File View
  • Reader View
  • Settings
  • Race List

Il bottone corrente è evidenziato. In TZERO spariscono Open RR e RR Live.

Settings

È la pagina principale di configurazione gara.

  • gestione file sorgente
  • timing point
  • blacklist
  • filtri From time / To time
  • simulated files

In TZERO il SYNC usa participants.csv, download/, timingpoints.json e sources.json.

File View

Mostra una riga per file/source.

  • status
  • totals
  • colonna Last 10 cycles
  • grafico minuto per minuto

Supporta Start time, Visible files e ordinamento manuale con frecce.

TP View

Vista a card per timing point.

  • ultimi 10 passaggi unici
  • totals aggregati
  • grafico del TP
  • bottoni Speaker e QR

Reader View

Vista a card per reader/device.

  • reader types: ICRON, UBIDIUM, FEIBOT
  • location: LAN, WAN, FS
  • lista file odierni, dimensione e orario
  • nome file verde se già collegato alla gara

Scan LAN censisce automaticamente i reader trovati. Scan WAN propone quelli attivi e lascia scegliere all'utente.

Speaker

Pagina pubblica tokenizzata, senza login, pensata per il secondo monitor o per un device in LAN.

Replica la card TP in formato full-screen e mostra anche il Team.

TZero

In TZERO ICRONIUM configura e governa la gara su filesystem.

  • timing point da config/timingpoints.json
  • sorgenti da config/sources.json
  • mirror dei file remoti dentro download/
  • delete sicuro: spostamento in deleted/ con timestamp
